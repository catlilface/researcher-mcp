# Researcher MCP

Researcher MCP is a Model Context Protocol (MCP) server designed specifically for scientific researchers. It provides a unified interface for accessing scholarly literature and research papers from multiple sources.

## Features

- **Tool-Agnostic**: Can be used with any MCP-compatible client environment
- **Multi-Source Support**: Currently supports arXiv and PubMed APIs
- **Research-Focused**: Designed with scientific researchers' needs in mind

## Supported Sources

- **arXiv**: Access to physics, mathematics, computer science, and related disciplines
- **PubMed**: Medical and life sciences research papers

## Getting Started

To use this MCP server, you'll need an MCP-compatible client. The server can be run independently and will provide access to the supported research databases.
